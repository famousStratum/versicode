Haritam

A web app for tracking and visualising outdoor activities. Import runs, hikes, and rides from GPX files or connect directly via Strava. Activities are rendered as interactive maps with elevation profiles and split breakdowns, and can be organised with custom routes and colour-coded tags.

Tech Stack

  • Next.js
  • TypeScript
  • React
  • Tailwind CSS
  • MapLibre GL
  • D3
  • Prisma
  • NextAuth
  • Resend
