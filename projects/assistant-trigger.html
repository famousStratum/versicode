AssistantTrigger

A minimal Android utility that maps a physical hardware button — like the Samsung Bixby key — to your system's default voice assistant. Instead of opening the full-screen assistant interface, it fires the system voice command intent directly, giving you a faster, more integrated experience. No UI, no permissions, no background processes — it launches, triggers the intent, and immediately exits.

Tech Stack

  • Kotlin
  • Android SDK
  • No UI
